# thoughtworks-academy-recruiting-system-automated-test

## Environment
###  JDK
###  gradle
###  nginx
###  Docker

## Initialize 
1. 启动 **nginx**
2. 启动 **Docker Container ( mysql, mongo, paper-api )**
3. 启动 **web-api** (test 环境)
4. 启动 **Jenkins**


## Run Test

```
# 初始化环境
./automation init

# 运行测试
./automation run
```
